Sobre o Projeto

Este projeto consiste na implementação de um carrossel interativo desenvolvido com HTML, CSS e JavaScript puro. A proposta é apresentar conteúdos de forma dinâmica e organizada, permitindo a navegação entre diferentes elementos de maneira fluida e intuitiva embasado em um projeto já existente encontrado na internet.

Tecnologias Utilizadas

HTML5 responsável pela estruturação semântica do conteúdo
CSS3 utilizado para estilização e criação de animações suaves
JavaScript aplicado na manipulação do DOM e controle das interações

Funcionalidades

Exibição de elementos em formato de carrossel com navegação entre itens
Animações realizadas com CSS utilizando propriedades como transition e transform
Troca dinâmica de classes para controle de estado dos elementos
Interface fluida com foco na experiência do usuário

Conceitos Aplicados

Separação de responsabilidades entre estrutura, estilo e comportamento
Manipulação de classes com uso de classList no JavaScript
Aplicação de animações performáticas com CSS
Organização de código visando legibilidade e fácil manutenção

Objetivo

O objetivo do projeto é praticar conceitos fundamentais do desenvolvimento front end, com foco na criação de componentes interativos sem o uso de bibliotecas ou frameworks externos, fortalecendo o domínio das tecnologias base da web.

Estrutura

O código foi estruturado de forma clara e modular, facilitando a manutenção, leitura e possíveis expansões futuras do projeto.
